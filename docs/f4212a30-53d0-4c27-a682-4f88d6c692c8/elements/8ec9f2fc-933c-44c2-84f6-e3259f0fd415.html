Som forretningsutvikler som gir innspill til regelverksutvikling ønsker jeg å se helheten på et utvalgt område slik at jeg kan vurdere hvordan lovendringsforslag vil slå ut på vårt område ()
Egentlig handler dette om helhetlig informasjonsforvaltning mer enn bare begrep - eks: Nytt pålegg - har vi dataene som vi trenger for å oppfylle pålegget? Hvis ikke, hvem har dem og kan vi gjenbruke? Hvilken kvalitet har de? Må vi utvikle noe eller endre våre prosesser for å etterkomme pålegget, osv.
